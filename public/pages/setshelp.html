Scoreboard

Click on a cell in the boxleague table to enter a score in the Scoreboard.

Results are entered into the scoreboard as set1 set2 [set3], with games entered for each player. Set3 is optional (see Rules), and can be entered either as a full set score or a 10 point tiebreak. A 10 point tiebreak can either be entered as a short tiebreak score e.g. 1:0 or a full tiebreak score e.g. 10:8.

Use the '+' and '-' buttons to increment or decrement the game count for each player in the set.

A played on date can be entered for the game. Specifying a played on date will allow you to review the history of your games at a later date. If a played on date is not entered then the game's scheduled date will be used.

Use the '+' and '-' buttons to select the games or tiebreak points (final set option). Note the number sequence for set one is W, C, 0, 1, ... 7 and for all other sets C, 0, 1, ... 99

Examples

  • 6:4 6:4 (straight sets win)
  • 6:3 2:6 10:8 (1 set and tiebreak win - full tiebreak score)
  • 6:3 2:6 1:0 (1 set and tiebreak win - short tiebreak score)
  • 6:3 2:6 6:4 (2 sets win - 3 full sets played)
  • W:0 (walkover - press the first set '-' button to select 'W')
  • 6:4 C:4 (concedes - press the '-' button to select 'C')

The  Save  button will only become available for a valid sets score.
