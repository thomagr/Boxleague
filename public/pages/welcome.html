Welcome

Use the menu links above to navigate to players, boxes, rules etc. In your box league click in the fixtures table to enter a result.

Enjoy!

Note: Fixture results can only be entered in the format, set1 set2 [set3], where sets are entered as games:games. Set3 is optional (see Rules), and can be entered either as a full set or a 10 point tie break.

E.g.
6:4 6:4 (straight sets win)
7:5 2:6 10:8 (1 set and tiebreak win - full tiebreak score)
6:3 2:6 1:0 (1 set and tiebreak win - short tiebreak score)
6:3 2:6 6:4 (2 sets win - 3 full sets played)

The following indicator assistance is available when entering a result:
 ?  - incomplete score, please complete entry
 !  - incorrect score, please edit entry
 *  - too many results, please remove incorrect entry
 Save  - score ready to be saved
